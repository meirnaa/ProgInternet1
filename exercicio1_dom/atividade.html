Diferenças entre innerText, innerHTML e textContent.

textContent: retorna todo o texto do elemento, ignorando estilos CSS e elementos HTML. Inclui até o conteúdo que está oculto na página.
innerText: retorna o texto visível para o usuário, ou seja, o que aparece visualmente na tela, levando em conta o CSS (como display: none, por exemplo).
innerHTML: retorna todo o conteúdo HTML interno do elemento, incluindo tags, atributos e texto. Pode ser usado também para inserir elementos HTML dinamicamente.
